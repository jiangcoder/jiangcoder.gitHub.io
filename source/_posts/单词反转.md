## 单词反转
```java
public String reverseWords(String s) {
    String[] strs = s.split("\\s");
    StringBuilder stringBuilder = new StringBuilder();
    for (int i = strs.length - 1; i >= 0; i--) {
        if (strs[i].equals(""))
            continue;
        stringBuilder.append(strs[i].trim() + " ");
    }
    return stringBuilder.toString().trim();
}
```
